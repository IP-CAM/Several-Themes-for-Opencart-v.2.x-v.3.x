SarahMarket - AliExpress Clone OpenCart Theme for Large Store

Inspired of AliExpress, SarahMarket is a OpenCart theme for OpenCart 3.x made for large stores, supermarket & groceries.

Theme has 3 variations: Clean, Red, and Emerald. Each variation has different creative homepages. Main features: Mega menu, slideshow, products by category, bulk order (wholesale b2b order), blog, custom category pages and more.

Compatibility: OpenCart 3.x

Many other features supported in this theme:

  • List and Grid Product displays
  • Mini dropdown cart
  • Write a review and rating for each product
  • Related Product with slider effect
  • Configurable side column position left or right
  • Show tag cloud of boss tags cloud module
  • Quick filter product by boss manufacturer module
  • Vertical category with multi level
  • Supporting OCmod
  • Supporting multi store
  • Supporting sticky menu, Animated effect
  • Supporting Testimonial module
  • Supporting RTL language
